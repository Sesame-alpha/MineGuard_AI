 MineGuard AI – Smart Mining Safety Dashboard

MineGuard AI is a real-time mining safety simulation dashboard that monitors environmental conditions such as temperature, gas levels, and rock stability. It calculates risk levels, visualizes trends, and provides AI-style safety predictions.

This project simulates how modern mining operations use control room dashboards and monitoring systems to ensure worker safety and operational stability.

--
Features

- Live sensor simulation (temperature, gas, rock stability)
- Risk scoring engine (safety logic system)
- Real-time safety status alerts
- Live risk trend graph (Chart.js)
- Interactive mine zone map with danger visualization
- AI-style safety prediction system

---

 How It Works

The system generates simulated sensor data every 2 seconds:

- Temperature (°C)
- Gas concentration (%)
- Rock stability (1–10 scale)

These values are processed by a risk engine that calculates a safety score:

- Low risk → Safe conditions
- Medium risk → Warning state
- High risk → Danger state (evacuation recommended)

The results are displayed in real time through:
- Dashboard status panel
- Live graph
- Mine zone map
